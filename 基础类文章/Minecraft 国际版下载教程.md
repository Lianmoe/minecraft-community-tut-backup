Minecraft 我的世界（中文社区大多直接使用英文称呼，本文也将使用英文名 Minecraft 进行讨论）是一个社区主导的游戏，玩家可以安装众多例如模组、资源包、光影等社区内容来极大丰富游戏体验，但这也使得入门 Minecraft 有了一定的门槛。本文将会以最简明的语言讨论如何安装 Minecraft 国际 Java 版

## 〇、什么是国际版、Java 版
*(各版本区别尚未进行详细编写)*  
Minecraft 分为国际版和中国版，而国际版又分为 Java 版和基岩版。实际上，在绝大多数地方所提到的皆是 Minecraft 国际 Java 版  
Minecraft Java 版仅能在电脑（Windows 或 苹果 MacOS）上游玩，本文也需在电脑上进行操作

## 一、安装 Java
### 为什么需要 Java
Minecraft 是一个使用 Java 语言编写的程序，只有电脑安装了 Java 语言的环境才可以运行游戏。

Java 不止有一个发行版本，因为他是开源的。有众多厂商基于开源的 OpenJDK 进行客制化，就像同样开源的安卓系统一样，在不同的安卓手机上有不同系统（MIUI、ColorOS 等）和不同安卓版本（11、13 等），而 Java 便也拥有了不同的发行版（甲骨文、Liberica、龙井等）和不同的版本号（Java 8/Java 11/Java 17等）。

每个发行版各有各的优势，而版本号则会影响能够游玩的 Minecraft 版本。因此，Minecraft 不像其他游戏下载前置运行环境那么直接，需要多一些操作，但是也并不复杂，下文中会详细介绍

### 安装 Java 是在安装什么
Java 的环境分为两种：JDK（Java Development Kit） 和 JRE（Java Runtime Enviroment）。 

其中，JRE 是用于运行 Java 程序的，一般为用户、玩家安装。JDK 是用于软件开发的，JDK 内也包含了 JRE，体量比较大，一般只为开发者安装。  
因此对于玩家来说，使用 JRE 即可满足需求，部分教程也会让下载 JDK，而 JDK 占用空间过大，本文就不推荐了

JRE 的各个发行版都会提供安装包（.msi 格式文件）或仅提供压缩包（.zip 格式文件），两者的差别只在于，安装包在将 JDK 解压在某个文件夹后，会自动配置 Java 路径的系统变量（用于在启动 Java 程序时告诉 Java 程序你的 JRE 解压在哪里，不过 Minecraft 的社区启动器会自动检测 Java，所以可以忽略）。运行安装程序或者解压压缩包便完成了 JRE 的安装，也就是 Java 的安装。 

### Java 不同版本号、不同发行版的差别

- Java 的版本号： 不同 Minecraft 版本所需要的 Java 版本不同。每一个 Minecraft 版本都有最低的 Java 版本要求，但越新的 Java 版本的性能和稳定性往往会更好，不过版本过高也可能导致兼容问题。经过权衡，我们推荐：1.0 ~ 1.16.5 使用 Java8，1.17 ~ 1.20.1 使用 Java17。部分版本，例如 1.16.5 可以使用 Java11 以获取更好的性能。部分模组或整合包也会强制要求更高的 Java 版本。
    - 因此我们推荐将 Java8、Java11、Java17 都安装上，由启动器自动选择
- Java 的发行版：各个发行版主要是功能和性能上的区别 *(各发行版差别尚未编写)*。
    - 根据 Minecraft 的实际需求，本文推荐使用 LibericaJRE 的 Full Version，且下文将使用 LibericaJRE Full Version 进行讨论

## 二、选择一个启动器下载并放在合适的地方
### Minecraft 启动器是什么
Minecraft 是一个十分特殊的游戏，它强依赖于社区，光影、地图、资源包、模组甚至模组加载器都是社区制作的。官方启动器并没有对这些社区内容进行支持，因此社区制作了自己的启动器（本文简称为社区启动器）来支援社区内容，并提供更多更便捷的功能。
十多年来，有数十个社区启动器诞生，目前市面上也有许多主流社区启动器可以选择。本文仅推荐 Hello Minecraft Launcher（简称 HMCL） 和 Plain Craft Launcher 2（简称 PCL2） 启动器。两个启动器皆能满足基本需求，也是目前国内最主流的启动器。由于两个启动器功能相近，本篇教程将仅讨论 PCL2 启动器（HMCL 和其他启动器也类似）  

PCL2 仅支持 Windows，HMCL 支持 Windows 和 苹果MacOS，因此****苹果用户请选择 HMCL**

## 三、初次使用启动器的工作：设置账户、开启版本隔离、建立游戏目录
### Minecraft 特殊的正盗版
Minecraft 并没有特别的防盗版操作。在启动器中，你可以自由的选择使用离线模式（即不登录）或正版登录（使用微软账户或 MOJANG 账户登录），且可随意切换，不会因此有其他游戏中的封号风险

正版登录和离线模式的游戏内容完全一致，离线模式如要联机，则使用模组便可正常进行。离线模式仅无法进入开启了正版验证的服务器

我们推荐任何人购买正版。但对于仅想尝鲜的新玩家和学生来说，也不妨先使用离线模式进行游玩，当有能力了或想要长期游玩时，再支持正版。正版购买教程详见

### Minecraft 的版本隔离
Minecraft 与其他游戏不同，他的模组加载器是由社区制作的，而模组加载器普遍无法直接在各个版本之间直接兼容模组，这就导致了模组作者若想让自己的模组能在某个版本运行，则需专门为其进行版本适配相关的开发。材质、光影、地图等也面临同样的版本兼容问题

而 Minecraft 原版的更新非常频繁，至今产生了 20 个大版本号（从 1.0 到 1.20），且更新跨度大（2011 年 ~ 2023 年），Minecraft 的众多社区内容（例如模组、材质、地图等等）大多仅能支持部分版本，因此玩家常常会为了某些社区内容而更换不同的版本

玩家也可能下载或者自己制作整合包，以追求不同的玩法，例如用于游玩原版的原版增强包、魔法主题的魔法包、科技主题的科技包，又或者是大杂烩的生活包。这也需要分开管理游戏文件

因此，社区启动器制作了版本隔离的功能，将每个版本的游戏文件皆隔离开，以方便管理


> ### 🔆推荐流程
>- **双击「Plain Craft Launcher.exe」启动 PCL2 启动器，点击头像上方的「离线」，并设置你喜欢的游戏昵称（仅限英文和下划线）**  
>- **再点击最上方导航栏中的「设置」，在「启动选项」中找到「版本隔离」并切换到 「隔离所有版本」**  
>- **最后在「版本选择」中点击「新建 .minecraft 文件夹」**

## 四、下载游戏
### Minecraft 的模组加载器
Minecraft 的模组加载器在安装在游戏版本上之后可以进行移除或更改（PCL2 启动器目前尚未开发该功能，这是 PCL2 启动器目前相较之下最大的缺陷，不过可以通过安装多个版本来解决，存档等可以直接转移）
### Minecraft 不同版本的选择
前面提到了，Minecraft 的版本会影响到地图、模组等社区内容的游玩。版本的选择需要根据情况加以考虑，以下是我们的建议：
- 如果你是新玩家，或者此前接触 Minecraft 不多：
    - 推荐游玩最新版本，且安装 fabric 加载器。玩任何有模组的游戏之前，都是建议不带模组先体验一遍原版的。这对于社区内容众多（实际上是最多，仅 curseforge 上便已有 13.9 万个模组）且原版内容并不多的 Minecraft 来说便更为有意义。安装了 fabric 之后，也可以方便此后安装光影和其他辅助模组（例如性能优化）
- 如果您仅想游玩原版：
    - 推荐游玩最新版本。Minecraft 很少有移除内容的更新，因此最新版往往是原版内容最多的版本。如果需要一些辅助类模组，例如背包一键整理、性能优化以及光影等，则可以选择安装 fabric 模组加载器，这是一种更轻量的模组加载器，其在原版优化类模组的表现远远好于 forge。
- 如果您想游玩模组：
    - 如果您已经有明确的想玩的模组了，那么请在 [mcmod.cn](https://www.mcmod.cn/) 中搜索，找到您想游玩的模组的条目，并查看其支持的版本。在这些受支持的版本中，如果有有 1.18.2 则优先选择
    - 如果您没有明确想游玩的模组，那么我们更推荐您游玩高版本（1.13+）。如果不知道选择哪个版本，那么目前来说 1.18.2 是最好的选择。
    - 本模块仅引导玩家选择版本，更多模组相关教程请查看 模组游玩教程（*尚未开发*）
